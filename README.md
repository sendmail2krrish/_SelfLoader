# _SelfLoader

Use _Self Loader to avoid include php classes.


## Uses of _SelfLoader

* Set your root directory in **config.json**
* Go to **bin** and run **php loader.php** using your terminal or power shell or cmd
* Add **autoload.php**

```
require_once "lib/_SelfLoader/autoload.php";
```
